# MVP-example
## Description
A sample project of iOS MVP architecture. Just table where you can add or delete users. A design is not the point of this project,. The main goal was to learn and practice MVP architecture on real project. 100% programmatic UI.
- MVP architecture
- UIKit 
- noStoryboards
